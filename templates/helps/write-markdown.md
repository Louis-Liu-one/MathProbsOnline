# 如何编写 Markdown

## 摘要

Markdown 是一种轻量级标记语言。我们的网站中所有的题目、题解、评论等需要或可能需要格式化排版的内容全部使用 Markdown 输入。本篇文档将帮助你编写自己的 Markdown 文本。

## 章节标题

在文本前部添加若干个 `#` 符号以插入标题：

Markdown    | HTML          | 渲染效果
------------|---------------|------------
`# H1`      | `<h1>H1</h1>` | <h1>H1</h1>
`## H1`     | `<h2>H2</h2>` | <h2>H2</h2>
`### H1`    | `<h3>H3</h3>` | <h3>H3</h3>
`#### H1`   | `<h4>H4</h4>` | <h4>H4</h4>
`##### H1`  | `<h5>H5</h5>` | <h5>H5</h5>
`###### H6` | `<h6>H6</h6>` | <h6>H6</h6>

上面的实例中 `#` 号与文本间的空格不可少。标题的内容建议使用纯文本，不要插入超文本内容，如公式等。

你也可以在文本的第二行插入若干个 `=` 或 `-` 使其成为一级或二级标题：
```
Heading 1
=========

Heading 2
---------
```
渲染效果与 `# Heading 1`、`## Heading 2` 相同。

## 段落与换行

使用空白行将文本分隔为段落：
```
这是第一段。

这是第二段。
```
这是第一段。

这是第二段。

在文本行末添加两个或以上的空格再键入换行符，可以实现段内换行。你也可以使用 `<br>` 标签换行：
```
这是第一行。<br>这是第二行。
```
这是第一行。<br>这是第二行。

## 粗体、斜体、粗斜体、删除线

使用 `**粗体**` 或 `__粗体__` 渲染**粗体**，使用 `*斜体*` 或 `_斜体_` 渲染*斜体*。两者嵌套使用则可以渲染***粗斜体***。也可以使用 `~~删除的内容~~` 渲染~~删除的内容~~。

请尽量使用 `*` 而减少使用 `_`，这是由于 Markdown 处理文本间的下划线时可能表现不同。例如，`壹_贰_叁` 得到“壹_贰_叁”，而 `壹 _贰_ 叁` 得到“壹 _贰_ 叁”，但这样会在汉字之间产生丑陋的空格。而 `壹*贰*叁` 可以直接得到“壹*贰*叁”而不会产生多余的空格。

## 引用

在单行或多行文本的每行前添加 `> ` 以创建引用：
```
> Here's to the crazy ones.
```

多段文本也可以创建引用：
```
> The misfits. The rebels. The troublemakers. The round pegs in the square holes.
>
> The ones who see things differently.
>
> They're not fond of rules. And they have no respect for the status quo.
```

## 有序列表、无序列表

按照如下格式创建有序列表：
```
1. 英语
2. 汉语
    1. 普通话
    2. 方言
3. 印地语
3. 西班牙语
```
1. 英语
2. 汉语
    1. 普通话
    2. 方言
3. 印地语
3. 西班牙语

或按照如下格式创建无序列表：
```
* 诗
    - 古体诗
    - 近体诗
* 词
* 曲
    + 散曲
    + 剧曲
```
* 诗
    - 古体诗
    - 近体诗
* 词
* 曲
    + 散曲
    + 剧曲

如你所见，列表可以嵌套，有序列表的编号也不必按照顺序。无序列表可以使用 `*`、`+` 或 `-` 字符创建。

要在一个列表项中插入段落、引用、代码块、图片、列表等元素，你需要将其缩进四个空格或一个制表符，例如：
```
1. 英语

    You can quote them, disagree with them, glorify or vilify them.<br>
    About the only thing you can't do is ignore them.<br>
    Because they change things. They push the human race forward.

2. 汉语

    > 生活就像海洋，只有意志坚强的人才能到达彼岸。
```
1. 英语

    You can quote them, disagree with them, glorify or vilify them.<br>
    About the only thing you can't do is ignore them.<br>
    Because they change things. They push the human race forward.

2. 汉语

    > 生活就像海洋，只有意志坚强的人才能到达彼岸。

你也可以使用 HTML 提供的 `<ol>`、`<ul>`、`<li>` 等标签创建列表。

## 表格

### 插入表格

使用如下格式创建表格：
```
| 字词 | 释义 | 出处 |
| --- | --- | --- |
| 安 | 怎么 | 安求其能千里也 |
| 卑 | 身份低微 | 先帝不以臣卑鄙 |
| 备 | 周全，详尽 | 前人之述备矣 |
```
| 字词 | 释义 | 出处 |
| --- | --- | --- |
| 安 | 怎么 | 安求其能千里也 |
| 卑 | 身份低微 | 先帝不以臣卑鄙 |
| 备 | 周全，详尽 | 前人之述备矣 |

可见，表格每行不需要严格对齐，符合格式即可。

### 表格列的对齐

在表线一行适当插入 `:` 以实现左对齐、居中、右对齐。
```
| 左对齐标题 | 居中标题 | 右对齐标题 |
| :------- | :-----: | -------: |
| 左对齐 | 居中 | 右对齐 |
```
| 左对齐标题 | 居中标题 | 右对齐标题 |
| :------- | :-----: | -------: |
| 左对齐 | 居中 | 右对齐 |

若要在表格中插入 `|` 字符，需要使用 `&amp;#124;` 转义。

## 行内代码与代码块

### 行内代码

使用 `` `code` `` 插入行内代码，渲染效果为 `code`。

若要在行内代码中使用单个或多个连续的 `` ` ``，则可以将最外层的 `` ` `` 替换为任意多个 `` ` ``，其数量应该大于代码中最长连续 `` ` `` 的数量。

你也可以使用 `<code>code</code>` 标签插入行内代码，渲染效果与 `` `code` `` 相同。

### 代码块

使用
````
```
def sigmoid(x):
    return 1/(1 + exp(-x))
```
````
插入代码块，效果为
```
def sigmoid(x):
    return 1/(1 + exp(-x))
```
你也可以在单行或多行文本的每行前添加四个空格或一个制表符以创建代码块：
```
    def sigmoid(x):
        return 1/(1 + exp(-x))
```
渲染效果与使用 ```` ``` ```` 相同。

## 分隔线

使连续三个以上的 `*`、`-`、`_` 自成一个段落，可以创建分隔线：
```
分隔线前面的内容。

---------------

分隔线后面的内容。
```
分隔线前面的内容。

---------------

分隔线后面的内容。
